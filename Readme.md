# ALLIB-The Collabrative Whiteboard

## How to run program

- Open two terminals in vs code
- In first terminal run:
```bash
cd Server
npm i
node index.js
```
- In second terminal run:
```bash
cd my-app
npm i 
npm run dev
```
-App will run on [https://localhost:5173]

## Basic structure

- my-app  folder contains  all UI related files 
- Server  folder contains  all backen related files 
- Both are  connected  with  axios
